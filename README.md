## 📋 Sobre o Projeto
Este repositório serve como um diário de bordo técnico e laboratório de códigos. Aqui, aplico conceitos de **Análise e Desenvolvimento de Sistemas (ADS)** para resolver problemas de **Física Clássica e Moderna**, utilizando **Python** como principal ferramenta de simulação.

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Bibliotecas Científicas:** NumPy (Álgebra Linear), Matplotlib (Visualização de Dados)
* **Conceitos de ADS:** POO (Programação Orientada a Objetos), Algoritmos de Integração Numérica (Euler)
* **Física:** Cinemática, Dinâmica, Eletromagnetismo

## 📂 Organização do Repositório
* `/simulacoes`: Scripts Python de fenômenos físicos (ex: MUV, Lançamento de Projéteis).
* `/estudos-teoricos`: Resumos e notas sobre as videoaulas de Física e Matemática.
* `/calculo-numerico`: Implementação de algoritmos para resolver EDOs (Equações Diferenciais Ordinárias).
